### WordPress Text to Speech Plugin – GSpeech TTS

[GSpeech](https://gspeech.io) is an advanced **WordPress Text to Speech plugin** that converts WordPress posts, pages, and WooCommerce product descriptions into natural-sounding **text-to-audio narration** in **70+ languages**. It automatically adds a modern **WordPress audio player** to posts and pages, allowing visitors to **listen to articles, blog posts, and product content** instead of just reading them.

Trusted by thousands of websites and **rated ⭐4.8/5 on WordPress.org**, GSpeech has received **200+ five-star reviews across [WordPress.org](https://wordpress.org/plugins/gspeech/#reviews) (150+ reviews), [Joomla Extensions Directory](https://extensions.joomla.org/extension/style-a-design/accessibility/gspeech/), AppSumo, and [Trustpilot](https://www.trustpilot.com/review/gspeech.io)**. GSpeech is also featured in the **AppSumo Customer Favorites collection**, with a **4.9/5 rating from 40+ verified reviews**. Since 2012, GSpeech has helped publishers improve **WordPress accessibility, engagement, and listening experience**, allowing visitors to **listen to blog posts, articles, and WooCommerce product content** while browsing or multitasking.

Audio is generated once using cloud voices and then cached for instant playback, making GSpeech one of the fastest **text-to-speech plugins for WordPress websites**. GSpeech can generate audio for **very long articles**, supporting content with **up to 7 hours of playback per article**. Adding audio versions of content helps improve SEO signals, grow organic traffic, increase time on page, and boost engagement, while allowing visitors to **listen instead of reading** and helping publishers **turn written content into audio** for easier consumption. Publishers use GSpeech to improve accessibility, increase content engagement, and boost conversions by turning written content into engaging audio experiences.

Unlike many text-to-speech solutions that rely on limited browser voices or require users to configure and pay for external API keys, GSpeech provides access to premium AI voices and advanced speech technologies with **simple all-in-one pricing**, **no API key setup**, and **one-click activation**. GSpeech uses **cloud-generated voices powered by Google technology**, processed on the GSpeech cloud infrastructure to deliver consistent audio playback across all browsers and devices. Commercial plans include premium AI voices powered by **[OpenAI (ChatGPT) text-to-speech](https://openai.com)** and **[Google Cloud Text-to-Speech](https://cloud.google.com/text-to-speech)**, including **Gemini Flash TTS**, **Gemini Pro TTS**, **Chirp3 HD**, **Neural2**, **Polyglot**, and **WaveNet**, along with speed and pitch controls, custom voice instructions, and the unique **Voices Vibes system with 100+ expressive voice styles**.

The integrated **[GSpeech Cloud Console](https://gspeech.io/dashboard)** lets users generate and manage **WordPress text-to-speech audio** directly from the admin panel. GSpeech is powered by a scalable cloud platform, ensuring fast and reliable **audio generation for WordPress websites at scale**. Commercial plans also support **real-time audio translation** in **70+ languages**, allowing visitors to translate and listen to content instantly. Even single-language websites can become **multilingual audio experiences**, helping publishers reach a global audience. The same GSpeech engine can also be integrated into **Shopify, Wix, Joomla, and custom HTML websites**, enabling publishers to add text-to-speech audio across multiple platforms.

== Installation ==

1. Go to the WordPress plugin installer and search for __"GSpeech"__.
2. Install the plugin or upload the plugin file downloaded from WordPress.org.
3. Activate the plugin through the __Plugins__ menu in WordPress.
4. After activation, you will see the __GSpeech__ menu in your WordPress admin panel.
5. Click __Activate Cloud Console__ (one click). The required widgets will be created automatically.
6. Insert a Full Page Player using the shortcode: __[gspeech]__.
7. Insert a Button Player using the shortcode: __[gspeech-button]__.
8. Insert a Circle Player using the shortcode: __[gspeech-circle]__.
9. The __Read Highlighted Text (RHT) Player__ will be activated automatically.

= 👉 How to insert a player in a post, page, or article? =
After activating the __Cloud Console__, the required widgets are created automatically.

__Shortcodes:__
1. Full Page Player – __[gspeech]__
2. Button Player – __[gspeech-button]__
3. Circle Player – __[gspeech-circle]__
4. Inline player – __{gspeech id=ID}text to speech{/gspeech}__ (ID = widget ID)

To edit player settings:  
Go to __WP-Admin → GSpeech → Cloud Console → Widgets__.

To automatically add players on multiple pages:  
Publish the __Multi-Page Widget__ inside the Widgets section.

You can also [contact us](https://gspeech.io/contact-us) and we will help configure it for your website.

= 👉 How can GSpeech improve WordPress accessibility? =
GSpeech provides multiple audio players for reading website content aloud. It supports three player types and more than __16 player themes__.

It also includes the __RHT Player (Read Highlighted Text)__, allowing visitors to listen to any selected text on the page.

= 👉 What is the Read Highlighted Text (RHT) Player? =
The __RHT Player__ improves accessibility by allowing visitors to listen to any selected text on your website.

It activates automatically when the plugin and __Cloud Console__ are enabled.

Configuration:  
Go to __WP-Admin → GSpeech → Cloud Console → Widgets → RHT Player__.

Available views:
• __View 1__ – Circle player with timeline, volume, and speed controls  
• __View 2 & 3__ (commercial plans) – include translation and text panels

See the [Screenshots Section](https://wordpress.org/plugins/gspeech/#screenshots) for examples.

= 👉 How to use Welcome Messages? =
After activating the __Cloud Console__, the Welcome Message widget is created but disabled by default.

To enable it:  
Go to __WP-Admin → GSpeech → Cloud Console → Widgets → Welcome Message__ and set __Status = Published__.

You can customize the message text, restrict it to specific pages using __Allowed URLs__, or configure it to play only once per session.

= 👉 Do you support themes like Divi? =
Yes. GSpeech works with most WordPress themes and builders including Divi. If you encounter compatibility issues, please [contact us](https://gspeech.io/contact-us).

= 👉 The player reads text that should be skipped – how can I fix this? =
Add the class __nospeech__ to elements that should be excluded from audio.

You can also use the __Exclude List__ in the Content tab to specify elements that should not be read.
